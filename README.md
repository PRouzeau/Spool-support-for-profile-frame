=======
Spool support and Bowden extruder support integrating simple filament sensor switch<br>
This is designed for aluminium profile frame (option for flat panels).<br>
Spool support have a flattened shape and is reclined to avoid spool balancing and have steady filament traction, in order to avoid filament tangling.<br>
